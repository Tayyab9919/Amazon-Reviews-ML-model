**# 📧 Amazon Reviews Sentiment Analysis (ML Project)**



**## 📌 Overview**

**This project focuses on classifying Amazon product reviews into sentiment categories using Machine Learning.**  

**The goal is to understand whether a review is positive or negative based on its text.**



**---**



**## 📂 Dataset**

**- Source: Amazon Reviews dataset (Excel format)**

**- Data was cleaned and converted into CSV for processing**



**---**



**## ⚙️ Tech Stack**

**- Python**

**- Pandas**

**- Scikit-learn**

**- TF-IDF Vectorizer**

**- Logistic Regression**



**---**



**## 🔄 Workflow**

**- Data Cleaning (removed unnecessary columns)**

**- Text Preprocessing (removed special characters, spaces)**

**- Data Annotation (labeling sentiment)**

**- Feature Extraction using TF-IDF**

**- Model Training (Logistic Regression)**

**- Model Evaluation**



**---**



**## 📊 Results**

**- - Accuracy: 72.5%**

**- Weighted F1-score: 0.61**

**- Generated classification report for performance evaluation**



**Note: Model performance is affected by class imbalance**
